## Introduction

ArrowOS is an AOSP/CAF based project started with the aim of keeping things simple, clean and neat.

## Installation Instructions
- Wipe System, Vendor, Data, Cache and Dalvik. Also, Format Data.
- Flash ROM
- Reboot
