## Introduction

crDroid is designed to increase performance and reliability over stock Android for your device also attempting to bringing many of the best features existent today!

## What's Working
- RIL
- Camera
- WiFi
- Storage
- 4G
- GPS
- VoLTE
- Offline Charging
- Battery LED

## Installation Instructions
- Make sure that all your zip ready for install
- TWRP - Wipe > Advanced Wipe >Wipe system, cache, dalvik,vendor and data
- Install Rom .zip
- Install Gapps .zip and magisk .apk(if needed)
- Reboot to system!
