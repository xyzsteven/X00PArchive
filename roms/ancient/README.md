## Introduction

AncientOS is Custom ROM based on AOSP that aims to provide a unique UI with some useful feautures for customization.

## Installation Instructions
- Wipe System, Vendor, Data, Cache and Dalvik. Also, Format Data.
- Flash ROM
- Reboot
