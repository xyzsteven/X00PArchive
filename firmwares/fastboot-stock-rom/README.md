# Fastboot Stock ROM for ASUS Zenfone Max M1 (X00P/X00PD)

## Installation Instructions
1. Connect your phone in Fastboot mode to PC with proper USB and Phone Drivers.
2. Download and extract the downloaded Fastboot ROM. Run 8917.bat or 8937.bat file according your Variant.
   - (For Snapdragon 425 Users: 8917.bat File)
   - (For Snapdragon 430 Users: 8937.bat File)
3. Wait for the process to complete.
